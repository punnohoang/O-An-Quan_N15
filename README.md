# oanquan-web

## Cau truc

- frontend: Next.js (chay `npm run dev` tren cong 3000)
- backend: Flask (chay `python app.py` tren cong 5000)

## Cach chay

### 1) Frontend

```bash
cd frontend
npm install
npm run dev
```

### 2) Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```
